タイマーやストップウォッチで勉強時間を記録すると、フクロウが成長し、 コインで服・家具・背景をカスタマイズできます。

Features

続けたくなる学習記録

⏱

勉強時間をかんたん記録

タイマーとストップウォッチで、始めたい時にすぐ記録できます。

🦉

フクロウが応援してくれる

記録後にやさしいメッセージが届き、勉強の積み重ねを見守ります。

🎀

コインで着せ替え

勉強時間でコインを集め、服・家具・背景を少しずつ増やせます。

💚

満腹度・好感度で表情が変化

お世話の状態に合わせてフクロウの表情や動きが変わります。

📊

Study Logで学習を見える化

最近の記録や週ごとの学習量を見て、無理なく振り返れます。

📱

スマホでも使えるWebアプリ

ブラウザから開けて、ホーム画面に追加すればアプリのように使えます。

How to use

使い方は3ステップ

1

科目を選んで勉強開始

英語、数学、資格勉強など、記録したい科目を選びます。

2

タイマー・ストップウォッチで記録

時間を決めても、とりあえず始めても、どちらでも記録できます。

3

コインを集めてフクロウを育てる

学習時間でコインを集め、自分だけのフクロウと部屋を作れます。

Install

スマホのホーム画面から使えます

FlowlはWebアプリなので、ブラウザから開いてホーム画面に追加できます。 いつものアプリのように、すぐ勉強を始められます。

iPhone

Safariで開く → 共有ボタン → ホーム画面に追加

Android

Chromeで開く → メニュー → ホーム画面に追加

勉強を、少しだけ育てる時間に。
